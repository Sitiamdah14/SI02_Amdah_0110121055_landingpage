Shared Hosting

Shared hosting adalah salah satu jenis hosting yang berbayar. Bagi Anda yang memiliki dana minim, maka shared hosting merupakan pilihan yang tepat. Pasalnya, layanan ini ditawarkan dengan harga yang sangat terjangkau. Shared hosting akan menempatkan beberapa situs website pada server yang sama. Dimana masing-masing website akan berbagi sumberdaya CPU dan RAM. Untuk website yang tidak begitu banyak pengunjung atau data, menggunakan layanan shared hosting tentu bisa membantu berhemat.

Virtual Private Server (VPS)

VPS sebenarnya hampir sama dengan jenis shared hosting. Hanya saja, sumber daya yang digunakan lebih tinggi kualitasnya. Menggunakan hosting server ini, maka klien bisa menggunakan seluruh sumber daya sekaligus menginstall aplikasi yang ada tanpa harus membaginya dengan klien lain. VPS sangat cocok untuk Anda yang memiliki website dengan banyak jumlah pengunjung.

Dedicated Server

Dedicated hosting memiliki konsep yang hampir sama dengan konsep shared hosting. Dimana hanya ada satu server saja yang digunakan. Namun, dedicated server memungkinkan untuk satu website memakai satu server. Tidak perlu berbagi dengan situs atau website lain. Dengan adanya hosting jenis ini, maka privasi website akan lebih terlindungi. Sebab tidak ada website lain yang menggunakan server yang sama dengan Anda. Tentu saja, performanya juga akan lebih baik dibandingkan dengan jenis server lainnya.
